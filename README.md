# Calculadora de notas
Projeto da disciplina de Metodologias Ágeis, esse projeto é feito para calcular a média ponderada das notas de um aluno, seguindo a fórmula (n1 + n2 + 2 * n3) / 4.

## Como executar?
- Primeiro, ative o ambiente virtual: <br />
WINDOWS: ```.venv\Scripts\activate``` <br />
LINUX: ```source .venv/bin/activate``` <br />
- Então, instale as dependências: <br />
```pip install -r requirements.txt``` <br />
- Por fim, rode os testes: <br />
```pytest -v``` <br />
