# Calculadora de notas
Projeto da disciplina de Metodologias Ágeis, esse projeto é feito para calcular a média ponderada das notas de um aluno, seguindo a fórmula (n1 + n2 + 2 * n3) / 4.

# Como executar
Primeiro, ative o ambiente virtual:
WINDOWS: ".venv\Scripts\activate"
LINUX: "source .venv/bin/activate"
Então, instale as dependências:
pip install -r requirements.txt
Por fim, rode os testes:
pytest -v
